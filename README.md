# Quer apostar quanto
 Desafio tecnico em Node + Express, TypeScript, Prisma (ORM), Postgres, Jest e Supertest

# Passos

  ```npm install```

  ```npx prisma init```

put your database credentials in .env file

  ```npx prisma db push```

  ```npm run dev:seed```

  ```npm run dev```
